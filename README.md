Swasthya Suraksha 🏥

AI-Powered Preventive Healthcare Monitoring System

🚀 Overview

Swasthya Suraksha is a preventive healthcare monitoring system designed to track vital health parameters in real-time and provide intelligent risk assessment. The system integrates wearable biometric sensors with a responsive web dashboard to enable early detection of potential health emergencies.

The goal of the project is to shift healthcare from reactive treatment to proactive prevention.

🎯 Key Features

✅ Real-time Heart Rate Monitoring

✅ Oxygen Saturation (SpO₂) Tracking

✅ Dynamic Risk Score Calculation

✅ Glassmorphism Health Dashboard

✅ Fully Responsive Design (Mobile-First)

✅ PWA Support (Installable Web App)

✅ Modern UI with Clean Medical Theme

✅ Sidebar Navigation with Smooth UX

🏗️ System Architecture

The system follows a modular IoT-based architecture:

Wearable Layer – Biometric sensors collect real-time data.

Mobile/Web Interface – Displays live metrics and health dashboard.

Cloud Layer (Conceptual) – Stores data and applies risk analysis logic.

Alert System (Future Scope) – Sends notifications during abnormal readings.

This structure ensures scalability and future AI integration.

🧠 Problem Statement

Many patients, especially elderly individuals and those with chronic conditions, lack continuous health monitoring outside clinical environments. Sudden health abnormalities often go unnoticed until critical stages.

Swasthya Suraksha aims to reduce response time and improve early detection through continuous monitoring and intelligent scoring.
